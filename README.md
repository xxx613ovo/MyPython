 淘宝用户行为分析项目 (2014版)

 1. 项目背景
基于阿里移动推荐算法大赛数据集 (2014年)，清洗百万级日志，分析用户行为模式，挖掘业务增长点。

 2. 技术栈
- Python: Pandas (数据清洗/ETL), Pyecharts (可视化)
- Tableau: 商业仪表板搭建
- 分析模型: 漏斗模型 (Funnel), RFM 用户价值模型

 3. 核心发现
- 流量规律: 每日 21:00-22:00 是用户活跃高峰，建议在此时间段推送营销活动。
- 转化漏斗: 点击到购买的转化率约为 2.3%，加购环节流失率较高。
- 用户价值: 识别出 20% 的高价值 (VIP) 用户群体，建议重点维护。
